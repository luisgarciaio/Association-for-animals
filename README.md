
# Project React Front-end

Project created solely as a frontend, with ReactJS, tailwindCSS, DeisyUI, animations, react router dom, LazyLoad and suspense


## Installation

Install asociacion-animales npm install

```bash
  npm install asociacion-animales
  cd asociacion-animales
  npm run dev
```
